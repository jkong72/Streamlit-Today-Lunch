## 개요
직업전문 교육 이수중 근처 지리나 식당 위치를 잘 몰랐기 때문에, 학교 근처의 맛집들을 쉽게 찾아보기 위해 만든 어플리케이션입니다.
수업 진행중에 배운 스트림릿 대시보드로 시각화를 진행하였으며, 근처의 위치를 찾기 위해 대형 서비스의 위치 API를 탐색하여 카카오 개발자 API를 사용하였습니다.

## 카카오 개발자 API (로컬)
카카오 로컬 API는 한번의 요청으로 인한 결과값이 제한되므로, 지정한 영역 내의 모든 장소를 찾아올 수 없습니다.
따라서 제한을 넘기는 응답 데이터가 있다면, 영역을 사분면으로 나눠, 각 사분면을 다시 탐색하여, 모든 결과를 합치는 방법으로 제작하였습니다.

## 지도 시각화와 아이콘 사용
지도는 pydeck을 사용하였으며, 한눈에 알아보기 쉽도록, 지점별 아이콘을 적용하였습니다.
